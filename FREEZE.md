# 灵枢 MVP 冻结版本

冻结日期：2026-06-25

## 本版本包含

- 5种人格原型（executor / creator / guardian / observer / explorer）
- 5个问题入口（sleep / anxiety / direction / relationship / energy）
- 20条镜像素材（三层组合：问题 + 行为 + 恢复等级）
- 评分系统（energy / recovery / stress）
- 恢复等级（good / light / medium / heavy）
- 最小版引擎（LingshuEngine.ts）

## 冻结期间不允许新增

- 社区 / 论坛
- 风水 / 流年 / 大运
- AI聊天 / 视频
- 会员体系
- 新人格
- 节气系统

## 解除冻结条件

- 完成 50 个真实用户测试
- 核心指标达标：
  - 首页 → 诊断 > 30%
  - 诊断完成率 > 60%
  - "有点准"反馈率 > 60%
