多语言配置已完成，请执行以下步骤：

1. 手动安装依赖：
   打开 CMD（不是 PowerShell），执行：
   cd /d D:\Projects\lingshu-app
   npm install next-intl

2. 本地构建测试：
   npm run build

3. 推送到 GitHub：
   使用 GitHub Desktop 提交并推送

4. Vercel 自动部署后，访问：
   https://lingshu-app-gules.vercel.app/zh
   https://lingshu-app-gules.vercel.app/en
   https://lingshu-app-gules.vercel.app/ja
   https://lingshu-app-gules.vercel.app/ko
   https://lingshu-app-gules.vercel.app/zh-TW

文件结构：
- i18n.ts (配置)
- middleware.ts (路由中间件)
- messages/ (5个语言文件)
- components/LanguageSwitcher.tsx (语言切换器)
- app/[locale]/ (所有页面已移入)
