## 新增模块

### 1. `lib/symbol/v3/graph.ts` — Growth Graph Engine
- 用户传播网络模型（UserNode：id/referrer/score/conversions/shares）
- GrowthGraph 类：增删查改 + localStorage 持久化
- getInfluenceScore()：加权影响力分（shares * 3 + conversions * 10 + score * 0.5）
- getTopNodes()：自动选超级传播者
- isSuperSpreader()：阈值 50
- getReferralDepth()：传播链深度追溯
- getNetworkHealth()：网络健康度报告

### 2. `lib/symbol/v3/factory.ts` — Autonomous Content Factory
- 6 角度变体：fear/identity/logic/contrast/reversal/insight
- 每变体带前缀 + 强度逐级递增
- pickByAngle() / pickHighestIntensity() / getFactorySummary()

### 3. `lib/symbol/v3/distribution_v33.ts` — Predictive Distribution Engine
- 渠道特征矩阵（tiktok/twitter/threads/linkedin/instagram/direct）
- predictBestChannel()：强度 + 角度双重匹配，输出冠军渠道 + 预期 CTR + 置信度
- distribute()：支持单输出或全排序

### 4. `lib/symbol/v3/index.ts` — 统一入口更新
- runV3Engine() 集成 V3.3：factoryProducts / distributionDecision / graph
- 导出所有 V3.3 模块
