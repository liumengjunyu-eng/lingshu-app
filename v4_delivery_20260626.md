**时间**：2026-06-26 11:45–12:02 GMT+8

## 新增模块

### `lib/symbol/v4/acquisition.ts` — Acquisition Brain（获客大脑）
- `AcquisitionBrain.predictChannelFit()`：根据用户画像（intensity/archetype/focus/emotion）自动选择最佳获客渠道
- `generateAcquisitionStrategy()`：输出渠道 + 策略描述 + 置信度
- `rankAllChannels()`：全渠道排序
- 支持：TikTok（高冲击情绪）/ Reddit（高分析力）/ Instagram（低认知负荷）/ Twitter / Threads / Google

### `lib/symbol/v4/content_brain.ts` — Content Brain（内容大脑）
- `generateAdaptiveContent()`：同一份 insight → 自动适配 4 种格式
  - viral_video：hook + 节奏 + 时长
  - seo_article：标题 + 结构 + meta description
  - social_post：文案 + 语调 + CTA
  - thread：Twitter 线程（自动拆分 insight 成 3-5 条推文）

### `lib/symbol/v4/conversion.ts` — Conversion Brain（转化大脑）
- `ConversionBrain.track()`：统一事件追踪（share=3 / signup=8 / pay=15 / return=5）
- `optimizePaywall()`：通过情感触发词评分选择最优付费墙文案（what happens / do nothing / your system 等 10 种触发器）
- `getConversionStats()`：转化数据统计

### `lib/symbol/v4/evolution.ts` — Evolution Brain（进化大脑）🔥
- `EvolutionBrain.mutateSystem()`：根据 conversion_rate / share_rate / channel_performance / return_rate 决定系统策略调整方向（increase / decrease / stable）
- `evolve()`：执行进化，更新 4 维配置 + mutation_cycle 版本号
- `getRecommendations()`：输出可读优化建议
- 完整 localStorage 持久化

### `lib/symbol/v4/index.ts` — 统一入口
- `createV4Engine()` 工厂函数
- `runCycle()`：一个调用跑完四大系统全闭环
