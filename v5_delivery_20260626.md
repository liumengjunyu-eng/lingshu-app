## 新增模块

### `lib/symbol/v5/revenue.ts` — Revenue Brain（收入大脑）
- `optimizePricing()`：根据 conversion / churn / avgRevenue / LTV 四维自动决策（降价/增加价值密度/扩张获客/引入订阅/稳定）
- `simulateRevenueScenario()`：低/中/高三档定价模拟
- `projectRevenue()`：按月收入预测模型

### `lib/symbol/v5/simulate.ts` — Market Simulation Engine（市场模拟）
- `simulateUserResponse()`：在发布之前预测 viral_probability / conversion_probability / retention_probability
- 内置关键词权重引擎（"not tired" +0.3, "system" +0.4, "pattern" +0.25）
- verdict 三档输出：scale / test / rework
- breakdown 因素分解（为什么会爆/为什么不爆）

### `lib/symbol/v5/marketing.ts` — Autonomous Marketing Engine（自动营销）
- `generateCampaign()`：一条产品信息 → 6 渠道全自动广告素材（TikTok / SEO / Twitter / Instagram / Threads / LinkedIn）
- `buildAd()`：每个渠道定制 hook / script / content / cta
- `pickBestChannel()`：根据用户画像选择最佳渠道组合

### `lib/symbol/v5/resource.ts` — Resource Allocation Engine（资源分配）
- `allocate()`：根据 ROI / CPA / retention / channelPerformance 自动决策（cut_budget / optimize_creative / improve_product / scale / rebalance_channels）
- `distributeBudget()`：智能预算分配（scale 模式：TikTok 35% / Google 25% / Twitter 15% / Instagram 15% / 测试 10%）
- cut_budget 模式自动降 90%，improve_product 模式保留 40%

### `lib/symbol/v5/index.ts` — 统一入口
