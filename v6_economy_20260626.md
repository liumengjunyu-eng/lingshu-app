# V6 Synthetic Economy 开发日志

## 工作内容

V6 三大系统全部从零实现：

1. **Company Swarm Layer** (`lib/symbol/v6/swarm.ts`) — 公司群体竞争/进化系统。8 种策略随机初始化，按 fitness 排序竞争，top 3 保留并变异繁殖，其余淘汰。

2. **Market Physics Engine** (`lib/symbol/v6/physics.ts`) — 市场物理引擎。demand/supply/attention/volatility 四维力学模型，可计算泡沫/崩盘/健康/入场时机，支持多步预测。

3. **Capital Flow Engine** (`lib/symbol/v6/capital.ts`) — 自动资本流动。幂律分配、再平衡、ROI 提取、撤资回收。

4. **UI 集成** — Synthetic Economy Running 状态行（结果页底部）。
